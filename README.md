# 密码管理器 (Password Finder)

一个基于Python 开发的轻量级本地账号密码管理工具，支持账号信息的新增、修改、删除、搜索，点击网站列可直接打开链接，点击密码列可复制密码到剪贴板。**当前版本【1.2】，更新于【2026/06/08】。**

## 功能特点

- 📝 本地存储：所有账号密码信息存储在本地`passwords.txt`文件，不联网更安全
- 🔍 快速搜索：支持按网站地址 / 用户名 / 标签模糊搜索
- 🏷️ 文字标签：支持为每个网站添加可搜索的文字标签（如"工作""邮箱""社交"），便于分类和检索
- 🌐 一键访问：点击网站地址列自动打开对应网页（自动补全 `http` / `https`）
- 📋 密码复制：点击密码列自动复制真实密码到剪贴板
- ✨ 界面简洁：清晰的表格展示，操作简单直观
- 🎨 美观样式：优化的配色和布局，提升使用体验

## 使用方法

- 直接点开 `PasswordFinder.exe` 即可。

### 基础操作

- **新增**：点击"新增"按钮，填写网站地址、用户名、密码、标签（可选），点击完成即可
- **修改**：选中要修改的行，点击"修改"按钮，修改信息后完成
- **删除**：选中要删除的行，点击"删除"按钮，确认后删除
- **搜索**：在搜索框输入关键词，实时筛选匹配的账号信息（支持按网站地址、用户名、标签搜索）
- **刷新列表**：重新加载本地存储的账号信息
- **打开网站**：点击表格中"网站地址"列的内容，自动在浏览器打开
- **复制密码**：点击表格中"密码"列的内容，自动复制真实密码到剪贴板

### 标签使用说明

- 新增或修改账号时，可在"标签"栏输入关键词，多个标签用逗号分隔，如 `工作,邮箱,重要`
- 搜索时输入标签关键词即可快速筛选出带有该标签的所有账号

## 数据存储格式

`passwords.txt`文件采用竖线`|`分隔字段，格式为：

```
https://www.baidu.com|baidu_user|baidu_pwd|搜索,工作
github.com|github_user|github_pwd|开发,代码
```

> 旧版本（1.1 及之前）的三字段格式 `site|user|pwd` 仍可正常读取，标签默认为空。

## 注意事项

1. 请妥善保管`passwords.txt`文件，建议定期备份
2. 本工具仅做本地存储，请勿存储超敏感密码信息
3. 若修改`passwords.txt`文件，需点击"刷新列表"生效
4. 仅支持Windows系统的密码复制功能（依赖 `clip` 命令）
